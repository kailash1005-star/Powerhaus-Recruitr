Recruitr

AI-powered recruitment lead generation platform. Full Next.js 15 + React 19 + TypeScript implementation following the Malo-Tech Outreach Design System.

Getting Started

  1. Install dependencies: npm install
  2. Run the development server: npm run dev
  3. Open http://localhost:3000 in your browser
Note: This is a static prototype build. All data is mocked with realistic loading states. The application demonstrates all 10 screens with full interactivity including navigation, filtering, and state management.
